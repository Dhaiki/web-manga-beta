Acerca de :

Dhaiki Mangas, Una página web donde encontraras todo tipos de mangas en formato de PDF o imágenes, totalmente gratis y seguros, con unos enlaces de descarga super fácil y rapidos, que contara con tres simple acortador, se recomemienda usar la extencion uBlock Origin en caso de publicidad excesiva.

Esta web es un pequeño proyecto que estoy iniciando como practica de mis estudios de Desarrollo Web (Frond-End), las metas que tengo con este proyecto es llegar a aprender Back-End e implementarlas y llegar a tener esta web al 100% funcional.

Si algo no llegara a funcionar bien se agradece dejar un comentario para aprender, mejorar y corregir la web.

Cualquier apoyo se agradece, sería una gran motivación para continuar con este proyecto.
